# 🎯 Number Guessing Game

My first interactive Python game! 👋 Developed by **Husniya** on her journey to building commercial games.

### 🎮 About The Game
This is a clean, text-based command-line game where the computer selects a random number between 1 and 100, and the player tries to guess it. The game provides smart feedback ("Too high" or "Too low") to guide the player.

### 💻 Technology Used
* **Language:** Python 3
* **Modules:** `random` (for generating secret numbers) and `time`.
* **Logic:** Loop logic, error handling with `try-except`.
